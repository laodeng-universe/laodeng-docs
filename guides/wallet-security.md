# Wallet Security

This guide describes basic security practices for using LAODENG Universe.

## Protect Your Seed Phrase

Your seed phrase controls your wallet.

- Never share it with anyone.
- Never enter it into an unknown website.
- Never send it through email or direct messages.
- Store backups offline in a secure location.
- Do not save it in public cloud storage or screenshots.

## Protect Your Private Key

A private key provides direct access to a wallet address.

Anyone who obtains it can control the assets in that wallet.

LAODENG UNIVERSE representatives will never ask for your private key or seed phrase.

## Verify Official Information

Only trust network information published through:

- https://laodeng.org
- https://github.com/laodeng-universe

Official network parameters:

- Chain ID: `20260001`
- RPC URL: `https://rpc.laodeng.org`
- Block Explorer: `https://scan.laodeng.org`

## Check Addresses Carefully

Before sending LDU:

1. Confirm the selected network is LAODENG Universe.
2. Check the entire recipient address.
3. Send a small test amount when appropriate.
4. Verify the transaction on the official block explorer.

## Avoid Impersonation

Be cautious of:

- Fake support accounts
- Unofficial airdrop links
- Messages promising guaranteed returns
- Requests to connect a wallet urgently
- Websites that imitate the official domain

Blockchain transactions are generally irreversible after confirmation.
