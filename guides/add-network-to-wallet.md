# Add LAODENG Universe to an EVM-Compatible Wallet

This guide explains how to manually add the LAODENG Universe mainnet to an EVM-compatible wallet.

## Official Mainnet Parameters

| Setting | Value |
|---|---|
| Network Name | LAODENG Universe |
| RPC URL | https://rpc.laodeng.org |
| Chain ID | 20260001 |
| Hex Chain ID | 0x13524a1 |
| Currency Symbol | LDU |
| Currency Decimals | 18 |
| Block Explorer URL | https://scan.laodeng.org |

## Manual Setup

1. Open your EVM-compatible wallet.
2. Open the network selector.
3. Choose **Add network**.
4. Choose **Add a network manually**.
5. Enter the official LAODENG Universe network parameters shown above.
6. Save the configuration.
7. Switch to **LAODENG Universe**.

## Verification

After adding the network, verify the following:

- Network Name: `LAODENG Universe`
- Chain ID: `20260001`
- Currency Symbol: `LDU`
- RPC URL: `https://rpc.laodeng.org`
- Block Explorer: `https://scan.laodeng.org`

## Security Warning

Only use network information published through the official website and GitHub organization:

- https://laodeng.org
- https://github.com/laodeng-universe

Never enter your seed phrase or private key into a website claiming to add the LAODENG Universe network.

LAODENG UNIVERSE representatives will never privately request your wallet credentials.
